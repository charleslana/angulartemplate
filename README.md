# Angulartemplate

## Instalar o ng

```bash
sudo npm install -g @angular/cli@17
```

## Instalar dependências

```bash
npm i
```

## Iniciar

```bash
npm start
```

## Gerar componentes

```bash
ng g c components/primary-input
```

## Gerar serviços

```bash
ng g s services/auth-guard
```

## Gerar páginas

```bash
ng g c pages/signup
```

## Gerar interceptores

```bash
ng g interceptor auth
```

## Criar projeto com o ng

```bash
ng new
```
